Tags usadas em html

  • H1: Titulo de primeiro nivel
  • H2: Titulo de segundo nivel.
  • H3: Titulo de terceiro nivel.
  • h4: Titulo de quarto nivel.
  • h5: Titulo de quinto nivel.
  • h6: Titulo de sexto nivel.
    • OBS: Somente vai ate o sexto nivel.
  • P: Usado na criação de Parágrafos
  • br: Usado para criar quebras de linha
  • & gt;: Cria um sinal de maior >
  • & Lt;: Cria um sinal de menor <
  • &#x + código do emoji: para adicionar um emoji. EX: & # x 1 f 6 2 d; .
  • Img: Para adiconar um imagem.
  • Lorem: gera um texto cheio de blá blá blá
  • Code: cria fontes monoespaçadas
  • Pré: organiza comforme o código é escrito. OBS: Poe ser combinado a outras tags como por exemplo a tag code.
  • Q: serve para colocar aspas quando for um citção simples.
  • Blockqute:Cria uma citação mais complexa.
  • abbr: mostra o significado de uma sigla.
  • ol: lista ordenadas.( sempre usar o li em seguida dentro do par de tags.)
  • Ul: Listas não ordenadas.( sempre usar o li em seguida dentro do par de tags.)
  • DL: Lista de definição.
  • Dt: termo da lista.
  • DD: Descriçao da lista.
  • A: cria um link.(Usado de varias formar

Tags Não semanticas.

  • B: deixa a palavra em negrito
  • I: deixa a palavra em Italico
  • Mark: Usado como marca texto
  • U: cria um texto sublinhado

Tags com semanticas

  • Strong: coloca a frase em negrito.
  • Em: deixa a frase em Italico.
  • small: deixa o texto um pouco menor.
  • del: cria um texto deletado.
  • INS: Cria um texto inserido
  • sup: cria texto sobescrito.
  • Sub: Cria um texto subescrito.

Tags em HTML Que usar sources ou pode ser combinadas com , source.

  • audio: serve para colocar audio no site. Usamo o parametro (source : src) (MP3, WAV, FLAC, AAC e WMA).
  • picture : sere para colocar imagens dinamicas.Usamos o parametro ( source: media type) (JPEG, PNG, BMP, GIF e TIFF)
  • video: serve para adiconar videos.Usamo o parametro (source: src) ( mp4,m4v,webm.obv).
    •                 
    Obs: todas essa tags dever ser usadas da seguinte forma:
   <Tag html>

    source

   <Tag html>

   widht: configura largura(Usado em imagem e video)

   controls: para aparecer controles.(Usado em audio e video)
   loop: apos o fim do video ou audio ele volta do inicio e repede.(Usado em audio e video)
   poster: usado para criar thumbanail(usado em video).

   preloud:metadata= carrega apenmas informaçãoes.( Usado em audio)

   preloud auto( padrão)= carrega o audio assim que a pagina é carregada(Usado em audio).

Prametros usados em css.

  • Style: inicia qual estilo voce quer colocar no seu site.
  • Fonte family: muda o fonte de uma letra:
  • color: Muda a cor da letra.
  • text align; justify; : organiza o seu conteudo e deixar todas a linhas iguais.
  • text align; Right: o texto fica a direita
  • text- decoration : underline; : texto sublinhado.
  • fonte-size: defini o tamhano da fonte de um texto
  • background-color: defini cor de fundo

Atributos REL e TARGET

  • Rel external: indica algo externo ( Usado em links>)
  • Rel next : Proxima pagina ( Usado em links>)
  • Rel prev: Pagina anterior ( Usado em links>)
  • Rel noflow: nao da importancia ao conteuo do link. ( Usado em links>)
  • Target _blank: abre uma nova guia. ( Usado em links>)
  • Target _self: abre na mesma guia. ( Usado em links>)

Iformações importantes para nao se esquecer.

todos as tags de img, audio e video, nao esquecer de adionar mais de um formato ou tamanho de arquivo. Existem tres formar de usar css. 1 forma: inline usa o css na mesma linha( não recomendado, apenas para situações especificas) 2 forma: Local/interna: usa o css no cabeçário do codigo. 3 forma: Externo= usa o css em uma pagina externo o mais usado e recomendado, use sempre que puder)
